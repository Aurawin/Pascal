PREQUISITES:

1.) Aurawin Social Computing Server and Console GUI must be installed.
2.) At least 1 AuraDisk must be configured for domain storage.

*********************************************************************
GETTING STARTED:

1.) To install Aurawin VDM, extract the contents of this zip file to a working folder
2.) Open your Aurawin SCS Console Application.
3.) Select or create a domain.
4.) Open the Aurawin File Manager for that domain.
5.) Select the /core/ folder and select File/Import menu item.
6.) When prompted select the ./core/ folder from the working folder.
7.) Select the /http/ folder and select File/Import menu item.
8.) When prompted select the ./http/ folder from the working folder.
9.) Optionally, import the latest keyword definitions for that domain.
10.) Optionally, import the latest purchase definitions for that domain.

Notice: It will take a few seconds for the keyword and template engines to 
register the changes made to system content.  Any keywords or purchases of 
the same name will be overwritten - so please review any changes and make
any necessary edits before importing keywords or purchases.
